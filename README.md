# Barnebys Auction Search

Technical assessment for the Senior Fullstack Developer role.

## Overview

This project is a mini auction search interface built with:

- Next.js
- TypeScript
- Node.js
- Express
- Tailwind CSS
- Vitest

The app allows users to browse, search, filter, sort, and inspect auction lots.

## Features

- Real-time search by title and description
- Filter by category
- Filter by country edition
- Responsive auction lot cards
- Estimate price display
- Auction house display
- Detail modal on card click
- Sort by estimate price
- Basic unit test for filtering logic
